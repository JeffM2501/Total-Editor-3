TILE FLAG:
1=Door(Horizontal)
2=Door(Vertical)
3=Switch
4=Teleporter
5=Secret Wall (ID = ORIENTATION, 0=RIGHT, 1=UP, 2=LEFT, 3=DOWN)
6=Nothing
7=Flat Wall (ID = ORIENTATION, 0 = HORIZONTAL, 1 = VERTICAL)
8=Locked Door(Horizontal, ID = KEYCARD, 0 = BLUE, 1 = BROWN, 2 = YELLOW, 3 = GRAY)
9=Locked Door(Vertical, ID = SEE ABOVE)
10=Disappearing Wall
11=Trigger (IF ID = 255 THEN IT MARKS A SECRET AREA)
12=Exit (IF ID > 0, ID IS THE MAP IT TAKES YOU TO, OTHERWISE IT'S THE NEXT MAP)
13=Conveyor Belt (ID = DIRECTION, SEE SECRET WALL)
